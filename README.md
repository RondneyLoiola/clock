# ⏰ CHRONOS - Modern Clock Application

A sleek and modern clock application with multiple time management features built with vanilla JavaScript, HTML, and CSS.

## 📋 About The Project

CHRONOS is a comprehensive time management web application featuring a clean, modern interface with four main functionalities:
- 🕐 **Real-time Clock** - Display current time and date
- ⏰ **Alarm** - Set custom alarms with quick presets
- ⏱️ **Timer** - Stopwatch functionality for tracking time
- 🌍 **World Clock** - Check time across different cities worldwide

## ✨ Features

- **Responsive Design** - Works seamlessly on desktop and mobile devices
- **Intuitive Navigation** - Easy-to-use sidebar navigation between features
- **Quick Presets** - Pre-configured alarm times for common activities
- **World Time** - Search and view time in different cities
- **Accessibility** - Built with ARIA labels and semantic HTML
- **Modern UI** - Clean, minimalist design with smooth transitions

## 💻 Getting Started

### Prerequisites

You only need a modern web browser to run this application.

### Installation

1. Clone the repository
```bash
git clone https://github.com/RondneyLoiola/clock.git
```

2. Navigate to the project directory
```bash
cd clock
```

3. Open `index.html` in your browser
```bash
# On Linux/Mac
open index.html

# On Windows
start index.html

# Or simply double-click the index.html file
```

## 🎯 How to Use

### Home (Clock)
- View current time in hours, minutes, and seconds
- See current day and date

### Alarm
- Choose from quick presets (5, 15, 20, or 45 minutes)
- Set custom alarm times
- Start alarm with play button

### Timer
- **Start** - Begin the stopwatch
- **Pause** - Pause the current time
- **Reset** - Reset timer to 00:00:00

### World Clock
- Search for any city in the world
- View local time and temperature
- See timezone information
